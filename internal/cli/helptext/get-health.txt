get-health — print canonical session health JSON

Usage:
  tmux-a2a-postman get-health [--session <name>] [--config <path>]
  tmux-a2a-postman get-health help

Flags:
  --session <name>        tmux session name (optional, auto-detected)
  --config <path>         Config file path (optional)

Output:
  JSON session health contract.
  Use nodes[*].visible_state for per-node state, queues for backlog counts,
  and compact for the compact display token.
