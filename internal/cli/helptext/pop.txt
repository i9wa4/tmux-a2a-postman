pop — read the next inbox message

Usage:
  tmux-a2a-postman pop
  tmux-a2a-postman pop --help

Output:
  Always JSON.
  {"status":"empty"}
  {"status":"message","id":"filename.md","message_id":"filename.md","from":"...","to":"...","timestamp":"...","body":"...","content":"...","unread_before":1,"remaining":0}

Fields:
  message_id              Stable message identity; id remains for compatibility
  body                    Parsed message body
  content                 Stored message file content exactly as written
  reply_slot_id           Exact reply slot id opened by reply-required mail
  fills_reply_slot_id     Exact reply slot id filled by this message
  reply_set_id            Aggregate reply slot set id when grouped
  obligation_id           Legacy alias for reply_slot_id
  satisfies_obligation_id Legacy alias for fills_reply_slot_id
  reply_to                Legacy message id this message resolves
  remaining               Unread messages remaining after this command
