Commands — detailed command reference
Use an explicit command. Bare `tmux-a2a-postman` prints usage; it does not start the daemon.

Default operator surface

send-heredoc
  Compose and deliver a message atomically with an explicit quoted heredoc.
  Output: JSON
  Usage:
    tmux-a2a-postman send-heredoc --to <node> <<'POSTMAN_BODY'
    literal $(not executed), `backticks`, and $HOME
    POSTMAN_BODY
  Flags:
    --to <node>          Recipient node name (required)
    --no-reply           Mark this message as not requiring a reply
    --reply-required     Mark this message as requiring a reply
    --reply-to <message> Message id for fallback closure or traceability
    --fills-reply-slot-id <id>
                         Exact reply slot id this message fills

send
  Legacy command name. Message body sending is disabled here.
  Use send-heredoc so the command surface makes quoted heredoc safety explicit.

pop
  Read and archive the oldest unread inbox message.
  Output: JSON
  Empty inbox: {"status":"empty"}
  Node is auto-detected from tmux pane title.

get-health
  Print canonical session health JSON for agents and scripts.
  Uses the current tmux session.

get-health-oneline
  Print compact all-session health for quick agent coordination.
  Shape: [0]🔷🟡:🟢 [1]🔴
  Window groups are colon-separated emoji runs with no literal window labels.

inspect-reply
  Inspect open reply-required work by message_id or reply_slot_id.
  Output: JSON
  Usage:
    tmux-a2a-postman inspect-reply --id <message_id-or-reply_slot_id>

version
  Print the build version JSON.
  Usage:
    tmux-a2a-postman version
    tmux-a2a-postman --version
  Output: JSON

Lifecycle and recovery

start
  Start the tmux-a2a-postman daemon.

stop
  Stop the running daemon for the current tmux session.
  Output: JSON
  Sends SIGTERM and waits up to 10 seconds for exit.
  Exits 0 if no daemon is running (idempotent).

help [topic]
  Show help overview or detailed topic page.
  Topics: messaging, directories, config, commands, start, stop, send-heredoc, send, pop, get-health, get-health-oneline, inspect-reply, version, help
