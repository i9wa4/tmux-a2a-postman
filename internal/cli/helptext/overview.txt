tmux-a2a-postman — A2A message routing daemon for tmux panes

AI agents send and receive messages. Each message is routed to the recipient
by the daemon. Use send-heredoc to send and pop to read.
Use an explicit command. Bare `tmux-a2a-postman` prints usage; it does not start the daemon.

Quick Start:
  1. Start daemon:    tmux-a2a-postman start
  2. Send a message:
     tmux-a2a-postman send-heredoc --to <node> <<'POSTMAN_BODY'
     message text
     POSTMAN_BODY
  3. Read next msg:   tmux-a2a-postman pop

Key Concepts:
  Node       An AI agent identified by its tmux pane title.
  Edge       A bidirectional routing rule between two nodes (configured in edges).
  Envelope   YAML frontmatter at the top of each message file:
               ---
               params:
                 from: <sender>
                 to: <recipient>
                 timestamp: <ISO 8601>
               ---

Default operator surface:
  send-heredoc               Send a message with an explicit quoted heredoc
  pop                        Read and archive the oldest unread inbox message
  get-health                 Print canonical session health JSON
  get-health-oneline         Print compact all-session health
  inspect-reply              Inspect open reply-required work by id
  version                    Print the build version JSON

Lifecycle and recovery:
  start                      Start the daemon (single-column TUI)
  stop                       Stop the running daemon for this tmux session
  Legacy and diagnostic helpers are internal, not CLI commands.

Messaging Protocol:
  send-heredoc --to <node> <<'POSTMAN_BODY' Send a message from quoted heredoc stdin
  pop                                        Read and archive oldest message
  get-health                                Inspect agent-readable session health
  get-health-oneline                        Inspect compact all-session health
  inspect-reply --id <id>                   Inspect an open reply slot by id
  version                                   Print the build version JSON
