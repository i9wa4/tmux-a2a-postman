stop — stop the running daemon

Usage:
  tmux-a2a-postman stop [--session <name>] [--timeout <secs>]
  tmux-a2a-postman stop help

Flags:
  --session <name>        tmux session name (optional, auto-detected)
  --config <path>         Config file path (optional)
  --timeout <secs>        Seconds to wait for daemon exit (default 10)

Output:
  Always JSON.
  {"status":"not_running","session":"review"}
  {"status":"stopped","session":"review","context_id":"...","pid":12345}
