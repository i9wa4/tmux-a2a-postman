get-status-oneline — print compact all-session status

Usage:
  tmux-a2a-postman get-status-oneline
  tmux-a2a-postman get-status-oneline --severity
  tmux-a2a-postman get-status-oneline --help

Output:
  [0]🔷🟡:🟢 [1]⚫

Severity Output:
  [0]needs_action:node=worker:input_required=1 [1]delivery_stuck:delivery:oldest_post_age=181s

  The default view prints compact visible-state marks.
  Use --severity for opt-in contextual severity tokens from compact_severity.
  A ? suffix marks inferred evidence, for example blocked?:node=worker.

Marks:
  ⚫ initial   no positive live evidence yet
  🔴 stale     previously known pane/session is stale
  🔷 pending   action-required inbound message is open
  🟡 waiting   outbound reply-required message is open
  🟢 ready     pane is live with no open action or wait

Screen progress fingerprints stay out of this compact view; use get-status
for nodes[*].screen_progress JSON evidence.
